## 🚀 Overview

**GitFace** is an interactive, plan-first profile generator built for modern developers. It moves beyond static templates, offering a high-fidelity drag-and-drop experience that ensures your GitHub profile stands out.

### Why GitFace?
- **Premium Aesthetics**: Designed with modern web standards (Glassmorphism, Dark Mode, Smooth Animations).
- **Plan-First Workflow**: Configure your sections, preview in real-time, and copy optimized markdown.
- **Brand Accurate**: Uses high-fidelity icons and official GitHub stat engines.

## ✨ Features

- [x] **Modular Sections**: Profile, tech stack, stats, socials, and extras — all as toggleable cards.
- [x] **Live Preview**: See your README exactly as GitHub renders it. Every keystroke updates the preview instantly.
- [x] **Drag to Reorder**: Arrange sections in any order with a smooth drag-and-drop interface.
- [x] **GitHub Themes**: Toggle between official GitHub Light and Dark themes to ensure perfect compatibility.
- [x] **Tech Badge Picker**: 80+ brand-accurate badges with a searchable picker.
- [x] **One-Click Export**: Copy optimized markdown or download the `.md` file directly.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router), [Tailwind CSS 4.0](https://tailwindcss.com/), [Shadcn/UI](https://ui.shadcn.com/)
- **State & Animation**: [Framer Motion](https://www.framer.com/motion/), [Zustand](https://zustand-demo.pmnd.rs/)
- **Icons & Branding**: Custom Octicons, [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com/)

## 🛠️ Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or pnpm

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/DJ-InfinityCoder/GitFace.git
   cd GitFace
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to see the builder.
