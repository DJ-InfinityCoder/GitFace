## 🚀 Overview

**GitFace** is an interactive, high-fidelity profile generator designed to help developers stand out. It moves beyond generic templates, offering a pixel-perfect drag-and-drop experience that ensures your GitHub profile looks premium on any device.

### Why GitFace?
- **Premium Aesthetics**: Built with high-fidelity SVGs, glassmorphism, and GitHub's native design language.
- **Real-Time Synergy**: Every change you make in the builder is reflected instantly in a 1:1 GitHub preview.
- **Data-Driven**: Integrated with advanced analytics for contribution streaks, top languages, and custom trophies.
- **Brand Accurate**: Includes a huge registry of official tech badges and social icons.

## ✨ Key Features

- [x] **Modular Section Builder**: Toggle and reorder Profile, Tech Stack, Stats, and Connect sections with ease.
- [x] **Premium Stat Cards**: Redesigned GitHub stats that feature segmented bars and grid layouts.
- [x] **Streak & Trophy Engines**: Showcase your consistency and achievements with interactive SVG cards.
- [x] **One-Click Markdown**: Instantly copy your optimized code or download the complete `README.md` file.
- [x] **Theme Synchronization**: Full support for both GitHub Light and Dark theme previews.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router), [Tailwind CSS 4.0](https://tailwindcss.com/)
- **UI Components**: [Shadcn/UI](https://ui.shadcn.com/)
- **State & Motion**: [Zustand](https://zustand-demo.pmnd.rs/), [Framer Motion](https://www.framer.com/motion/)
- **Analytics**: [GitHub API](https://docs.github.com/en/rest)
- **Icons**: Custom Octicons, [Lucide React](https://lucide.dev/), [Developer Icons](https://github.com/tandpfun/skill-icons)

## 🏗️ Getting Started

### Prerequisites
- Node.js 20.x or later
- npm, pnpm, or bun

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/DJ-InfinityCoder/GitFace.git
   cd GitFace
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Spin up the builder**
   ```bash
   npm run dev
   ```
   Navigate to `localhost:3000` and start building!
